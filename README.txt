SI.FA. QC v2

Contenuto:
- app web installabile (PWA)
- logo SI.FA.
- 21 pallet fissi
- 3 foto per pallet
- codice colore manuale
- stima automatica base
- stampa / PDF
- archivio locale nel browser

Come provarla:
1. Estrai lo ZIP.
2. Apri index.html da PC per prova rapida.
3. Per iPhone: questa app si installa bene quando viene pubblicata online su un dominio o hosting statico.
4. Una volta online, apri il link con Safari e usa "Aggiungi alla schermata Home".

Nota:
Questa è la versione 2 come app PWA. Per una versione aziendale vera multi-dispositivo serve hosting + database cloud.
